React

is a JavaScript library for building user interfaces.

Initially it was created on Facebook for internal use.

React...

  • => created to improve UI development

  • => allows to create apps that can change data without reloading the page

  • => provides ‘components’ development concept

  • => makes DOM updates to be very fast with Virtual Dom

Components concept

Put the components together

=>

Tree data structure that represents the UI of your application.

Virtual DOM

React updates its virtual DOM first.
Then updates only the object that have changed in the real DOM.

JSX

JavaScript Extension is a React extension which allows writing JavaScript code that looks like HTML.

with JSX



const name = 'React';
const element = <h1>{name} is cool!</h1>

ReactDOM.render(
	element,
	document.getElementById('root')
);

without JSX



const element2 = React.createElement;

ReactDOM.render(
	element2('h1', null, 'Hello, world!'),
	document.getElementById('root')
);

conclusion

  • => React library makes development easier

  • => improves the performance of dynamic websites

  • => provides ‘components’ concept for building highly interactive UI's
